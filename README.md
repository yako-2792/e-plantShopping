# e-plantShopping

## Paradise Nursery - Shopping Application

This is the final project for the React course: **e-plantShopping**, a front-end 
shopping application for Paradise Nursery, a houseplant store.

## Overview

The e-plantShopping application allows users to:

- Browse a variety of houseplants organized by categories
- Add plants to a shopping cart
- View the total number of items and total cost in the cart
- Adjust quantities (increase/decrease) of items in the cart
- Remove items from the cart
- Navigate between the landing page, product listing, and shopping cart

## Pages

1. **Landing Page** - Welcome page with company information and a "Get Started" button
2. **Product Listing Page** - Displays houseplants grouped by categories with "Add to Cart" functionality
3. **Shopping Cart Page** - Shows selected items, totals, and checkout options

## Technologies Used

- React
- Redux (Redux Toolkit)
- CSS
- GitHub Pages (for deployment)
